Event API Documentation

Welcome to the documentation for the Event API. This API allows you to create, read, update, and delete events.

Endpoints

GET /api/v3/events

Returns a list of all events. you can provide event type, limit and page too.
example, /api/v3/events?type=latest&limit=5&page=1

GET /api/v3/events/:id

Returns the event with the specified ID.

POST /api/v3/events

Creates a new event.

Parameters

  • type: The type of the event.
  • name: The name of the event.
  • tagline: A short tagline for the event.
  • schedule: The schedule for the event.
  • description: The description of the event.
  • moderator: The name of the moderator for the event.
  • category: The category of the event.
  • sub_category: The sub category of the event.
  • rigor_rank: The rigor rank of the event.
  • image: An image for the event.

PUT /api/v3/events/:id

Updates the event with the specified ID.

Parameters

  • type: The type of the event.
  • name: The name of the event.
  • tagline: A short tagline for the event.
  • schedule: The schedule for the event.
  • description: The description of the event.
  • moderator: The name of the moderator for the event.
  • category: The category of the event.
  • sub_category: The sub-category of the event.
  • rigor_rank: The rigor rank of the event.
  • image: An image for the event.

DELETE /api/v3/events/:id

Deletes the event with the specified ID.
